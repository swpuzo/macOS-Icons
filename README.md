# macOS Icons

A meticulous collection of system and third-party application icons designed specifically for macOS. This pack brings a unified, dark-themed aesthetic to your Dock and Launchpad, strictly following Apple's squircle design guidelines while introducing a sleek, dark mode style.

## Features

- **Squircle Form Factor:** Every icon perfectly matches the standard macOS rounded rectangle geometry for a native look.
- **Dark Mode Optimized:** Designed with deep, rich background plates that are easy on the eyes and perfect for dark setups.
- **High Resolution:** All assets are provided in high quality (up to 512x512) ensuring crisp rendering on Retina displays.
- **Comprehensive Coverage:** Includes essential Apple system apps and popular third-party tools tailored into the macOS squircle shape.
